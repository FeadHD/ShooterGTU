UI Manager Documentation

Overview

The UI Manager is responsible for creating, managing, and updating all user interface elements in the game. It handles health bars, score displays, messages, and debug information.

Core Features

  • Health and stamina bar management
  • Score and collectibles display
  • Start/end game messages
  • Debug information overlay
  • UI animations and transitions

Key Methods

UI Setup

setupUI() {
    // Create UI container
    this.container = this.scene.add.container(0, 0);
    this.container.setDepth(1000);
    
    // Create health bar
    this.createHealthBar();
    
    // Create score display
    this.createScoreDisplay();

Health Bar Management

Score Display

updateScore(score) {
    if (this.scoreText) {
        this.scoreText.setText('Score: ' + score);
    }

Usage Examples

// Initialize UI Manager
const uiManager = new UIManager(this);

// Update health display
uiManager.updateHealthBar(currentHealth);

// Update score
uiManager.updateScore(playerScore);
