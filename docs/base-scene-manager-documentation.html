Base Scene Documentation

Overview

BaseScene serves as the foundation for all game scenes, handling core functionality like physics setup, music, player creation, and game state management.

Core Methods

create()

Main initialization method that sets up the entire scene in a structured way.

create() {
  // 1. Basic Setup
  this.#disableRightClickMenu();
  const { width, height } = this.#setupCanvasDimensions();
  this.#initializeManagers();

  // 2. Audio & Input
  this.#setupMusic();
  this.#enableKeyboardInput();
  // ... and more initialization steps
}

Private Helper Methods

#disableRightClickMenu()

Disables browser context menu for better game experience

#disableRightClickMenu() {
  this.input.mouse.disableContextMenu();
}

#setupCanvasDimensions()

Gets game canvas dimensions for proper scaling

#setupCanvasDimensions() {
  return {
    width: this.cameras.main.width,
    height: this.cameras.main.height
  };
}

#setupMusic()

Initializes background music with volume settings

#setupMusic() {
  const bgMusic = this.sound.add('bgMusic', { loop: true });
  const musicVolume = this.registry.get('musicVolume') || 1;
  bgMusic.setVolume(musicVolume);
}

Public Methods

getSpawnHeight()

Returns the correct spawn height for entities

getSpawnHeight() {
  return this.groundTop - 16;
}

createUI()

Creates and initializes the game UI

createUI() {
  this.gameUI = new GameUI(this);
}

Best Practices

  • Always call super.create() first in child scenes
  • Use private methods (#) for internal scene logic
  • Use public methods for functionality needed by other scenes
  • Keep methods focused on single responsibilities
  • Use the scene state for tracking game progress
  • Handle errors appropriately in each method

Example Usage

class MyGameScene extends BaseScene {
  create() {
    // Call parent create method first
    super.create();

    // Add custom scene setup
    const spawnY = this.getSpawnHeight();
    // Create enemies, items, etc.
  }
}
